LineAge v1 supports:

- multiple trees per user
- one owner per tree
- people with first name required
- optional last name, gender, birth date, death date, birth place, biography
- relationships: parent/child/spouse
- internally relationships are PARENT or SPOUSE
- max 1 spouse per person
- max 2 parents per child
- no parent-child cycles
- disconnected family groups are allowed
- no uploads
- no edit sharing
- public view-only sharing later in v1
- The same pair of people cannot have more than one relationship between them, regardless of type or direction.
